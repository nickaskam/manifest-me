Vesting & Process

How to do this properly

  • Agree before any external money. Once investors are in, changing the cap table is painful and signals internal conflict.
  • 4-year vesting, 1-year cliff for all founders — standard in every term sheet. Protects everyone if someone leaves early.
  • The honest framework: ask each founder privately "what split feels fair?" — if answers are within 5%, go equal. If not, use Option A or C.
  • Use Carta or Pulley to model the cap table before signing anything. Both free at early stage.
  • Consider a dynamic equity model (Slicing Pie) if commitment levels are still being determined — each founder earns equity proportional to time contributed.
  • Get an attorney to draft the actual agreements — not expensive at this stage (~$1,500–3,000 for a standard founder package).

Founder Alignment Checklist

Before raising a dollar, agree on these

  • Equity split — get this in writing before any external money. Use Carta or an attorney.
  • Vesting schedule — 4-year, 1-year cliff for all founders. Non-negotiable for institutional investors.
  • Incorporation — incorporate as a Delaware C-Corp before fundraising. Investors expect it.
  • Founder titles — align on titles before investor meetings. Inconsistent titles signal misalignment.
  • Commitment level — who is full time now vs. at what milestone? Get this explicit.
  • Subscription price — set before App Store launch. Hard to raise prices on existing users.
  • Target market — consumer only, or does B2B (corporate wellness) get pursued in parallel?
  • IP assignment — make sure all founders have signed IP assignment agreements assigning work to the company.
